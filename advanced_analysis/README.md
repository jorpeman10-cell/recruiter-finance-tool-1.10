# 猎头进阶财务分析工具 - 三速差模型

> **猎头公司的生死线 = 职位流速度 × 顾问产能 × 现金周转**

这是一个独立于基础版本的高级财务分析工具，专注于单职位边际贡献和现金流日历，帮助老板实时掌握公司经营的健康度和生死线。

---

## 核心概念：三速差模型

### 1. 职位流速度 (Position Velocity)
衡量职位从发布到回款的流转效率

| 指标 | 说明 | 健康范围 |
|------|------|----------|
| 平均成单周期 | 发布到offer的天数 | < 60天 |
| 平均回款周期 | 入职到回款的天数 | < 45天 |
| 阶段转化率 | 各阶段推进效率 | > 30% |

### 2. 顾问产能 (Consultant Capacity)
衡量顾问团队的人效产出

| 指标 | 说明 | 健康范围 |
|------|------|----------|
| 产能利用率 | 在操作职位/最大容量 | 70-85% |
| 人均月成单 | 每人每月成单数 | > 1.5单 |
| 单职位边际贡献 | 每个职位的真实盈利 | > 0 |

### 3. 现金周转 (Cash Turnover)
衡量公司的现金流健康度

| 指标 | 说明 | 健康范围 |
|------|------|----------|
| 现金跑道 | 现金可支撑天数 | > 90天 |
| 周消耗率 | 每周固定支出 | < 周流入 |
| 资金缺口 | 预期负现金流 | = 0 |

## 功能模块

### 📊 三速差仪表盘
- **综合健康度评分**：基于三个维度的加权评分
- **职位流速度看板**：在途职位、阶段分布、周期分析
- **顾问产能监控**：利用率、瓶颈识别、人均产出
- **现金健康度**：跑道预测、缺口预警、周转效率

### 💹 单职位边际贡献模型
每个职位的真实盈利/亏损核算：

**边际贡献 = 实际回款 - 直接成本**

直接成本包括：
- 顾问成本（月工资×倍数法分摊）
- Sourcing成本（广告、渠道等）
- 面试成本（差旅、会议室等）

**关键指标：**
- 单职位边际贡献（MC）
- 日均边际贡献（MC/天）
- 盈利职位占比
- 顾问MC排名

### 📈 Forecast预测分析
分析在途单（预测职位）的预期价值与止损决策：

**核心逻辑：**
```
加权预测收入 = 预计佣金 × 阶段成功率
预期边际贡献 = 加权预测收入 - 已累计投入成本
ROI = 预期边际贡献 / 已投入成本
```

**阶段成功率映射（来自ForecastAssignment）：**
| 阶段 | 成功率 |
|------|--------|
| 简历推荐 | 10% |
| 客户一面 | 25% |
| 客户三面 | 40% |
| 终面 | 50% |
| offer | 80% |
| 入职 | 100% |

**决策建议：**
- ✓ 预期MC为正 → 继续投入
- ✗ 预期MC为负 → 考虑止损

### 📅 现金流日历
按日动态预测未来现金流：

- **日历视图**：每日流入/流出/余额预测
- **趋势图表**：现金流走势可视化
- **预警提示**：提前识别资金缺口日期
- **假设分析**：调整参数看影响

### 💡 决策建议
基于数据的智能建议：

- 客户策略优化
- 人员配置调整
- 现金流管理建议
- 假设分析工具

## 成本计算方式（重要）

由于无法精确统计每个顾问在每个职位上的工时，本工具提供**多种成本计算方式**：

### 🌟 1. 月工资×倍数法（推荐，符合内部核算）

这是最符合猎头公司内部核算的方式：

```
职位顾问成本 = 月工资 × 倍数 × (周期天数/30) ÷ 同时在操作职位数
```

**倍数构成（您使用3倍）：**
| 项目 | 倍数占比 | 说明 |
|------|----------|------|
| 基本工资 | 1.0× | 顾问月基本工资 |
| 奖金 | 1.0× | 成单奖金（约等于一倍工资） |
| 社保公积金 | 0.4× | 公司缴纳部分 |
| 固定成本均摊 | 0.3× | 房租、行政等 |
| 运营费用均摊 | 0.3× | 系统、品牌、管理等 |
| **合计** | **3.0×** | 您内部使用的成本线 |

**示例计算：**
- 顾问月基本工资：15,000元
- 倍数：3倍
- 同时在操作职位数：6个
- 职位周期：60天

```
月总成本 = 15,000 × 3 = 45,000元/月
单个职位月成本 = 45,000 ÷ 6 = 7,500元/月
该职位顾问成本 = 7,500 × (60/30) = 15,000元
```

**配置参数：**
- 顾问月基本工资：根据职级设置（顾问10-15K，经理20-30K）
- 倍数：您内部使用3倍
- 同时在操作职位数：根据实际（通常5-8个）

**👥 从顾问数据表自动提取工资（推荐）**

您可以上传顾问数据表，工具将自动提取每位顾问的工资用于成本核算：

| 字段 | 说明 | 必填 | 示例 |
|------|------|------|------|
| name / 姓名 / 顾问 | 顾问姓名 | ✅ | 张三 |
| base_salary / 基本工资 / 月薪 | 月基本工资 | ✅ | 15000 |
| avg_positions / 在操作职位数 | 同时在操作职位数 | ❌ | 6 |

**CSV示例：**
```csv
姓名,基本工资,在操作职位数
张三,15000,6
李四,20000,5
王五,18000,7
```

**优势：**
- ✅ 不同顾问可以使用不同工资（如初级顾问12K，资深顾问20K）
- ✅ 自动根据顾问姓名匹配职位，精确计算每个职位的成本
- ✅ 无需在侧边栏手动设置统一工资
- 同时在操作职位数：根据实际（通常5-8个）

### 2. 周期天数法
按职位持续时间估算成本：
```
顾问成本 = 周期天数 × 日均顾问成本
```
- **日均顾问成本**：建议设为 300-800元/天

### 3. 佣金比例法
按佣金收入的比例估算成本：
```
顾问成本 = 佣金金额 × 成本比例
```
- **成本比例**：建议设为 25-35%

### 4. 仅录入成本
只使用 Excel 中明确录入的成本字段（广告、面试等），不计算顾问人力成本。

## 关键阈值配置

在侧边栏可调整：

### 成本计算配置
- **成本计算方式**：智能组合（推荐）/ 周期天数法 / 佣金比例法 / 仅录入成本
- **日均顾问成本**：默认 500元/天（用于周期天数法）

### 其他配置
- **月度固定费用**：默认 50000元
- **现金流预警天数**：默认 30天
- **产能上限**：默认 8个职位/顾问

## 预警机制

### 🔴 紧急预警
- 30天内出现资金缺口
- 超过5个职位边际贡献为负
- 顾问产能利用率超过95%（超负荷）

### 🟡 警告预警
- 职位周期超过90天未结案
- 顾问产能利用率低于30%（闲置）
- 现金跑道不足60天

### ✅ 健康状态
- 所有指标在正常范围内
- 现金跑道充足
- 产能利用合理

## 决策应用场景

### 场景1：是否继续投入某客户？
查看该客户历史职位的**平均边际贡献**，如果为负或低于平均水平，考虑减少投入。

### 场景2：是否需要招聘新顾问？
查看**产能利用率**，如果超过85%的顾问占比高，且边际贡献为正，考虑扩编。

### 场景3：现金流紧张怎么办？
查看**现金流日历**，找到资金缺口日期，提前催收应收账款或控制支出。

### 场景4：如何提高盈利？
查看**单职位边际贡献**排名，分析高MC职位的共性，复制成功模式。
