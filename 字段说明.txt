=====================================
    数据模板字段说明
=====================================

【成单数据】字段说明：
-------------------------------------
支持中英文列名，以下列名都可以识别：

必填字段（以下列名均可）：
- deal_id / 成单编号 / 编号
- client_name / 客户 / 客户名称
- candidate_name / 候选人 / 姓名
- position / 职位
- consultant / 顾问 / 负责顾问
- deal_date / 成单日期 / 日期
- fee_amount / 佣金 / fee
- payment_status / 回款状态
- actual_payment / 实际回款 / 回款

可选字段（以下列名均可）：
- annual_salary / 年薪 → 不采集可留空
- fee_rate / 费率 → 不采集可留空，默认20%
- payment_date / 回款日期 → 未回款可留空
- prior_year_collection / 上年遗留回款 / 遗留回款
  * 去年offer/开票但今年回款的金额
  * 算入本年收入计算
  * 如果没有则填0或留空


【顾问数据】字段说明：
-------------------------------------
必填字段：
- name: 顾问姓名
- base_salary: 真实底薪（元/月），用于真实成本计算
- join_date: 入职日期
- team: 所属团队/部门
- is_active: 是否在职（True/False）
- monthly_kpi: 月度业绩目标（元）

可选字段：
- internal_base_salary: 内部核算基本月薪（元/月）
  * 用于给顾问展示的盈亏分析
  * 如果为空，则使用 base_salary
  * 计算公式：顾问成本 = internal_base_salary × 3 × 1.2
  * 3倍包括：工资、社保公积金、奖金、增值税、营运成本（支持部门、房租、办公等）
  * 1.2 = 20%利润目标


【费用数据】字段说明（自定义模板）：
-------------------------------------
必填字段：
- 日期: 费用发生日期（格式：YYYY-MM-DD）
- 费用类型: 人员工资/租金/差旅/招待/律师费/财务费/猎头系统费用/其他
- 金额: 费用金额（元）

可选字段：
- 部门: 费用归属部门
- 备注: 费用说明

费用类型说明：
- 人员工资：运营人员工资（不含顾问）
- 租金：办公室租金
- 差旅：出差交通住宿等
- 招待：客户宴请等
- 律师费：法律咨询、合同审查
- 财务费：审计费、税务咨询
- 猎头系统费用：系统订阅、软件费
- 其他：不属于以上类别的费用


=====================================
    两种分析模式说明
=====================================

【真实数据模式】
-------------------------------------
📊 **需要的数据**：
  - 成单数据（必填）
  - 顾问数据（可选，用于计算人力成本）
  - 费用数据（可选，用于统计运营成本）

📈 **计算方式**：
  - 顾问成本 = 真实底薪 + 提成
  - 运营成本 = 真实费用数据
  - 总成本 = 顾问成本 + 运营成本

🎯 **用途**：公司内部真实财务分析

【内部核算版本（给顾问展示）】
-------------------------------------
📊 **需要的数据**：
  - 成单数据（必填）
  - 顾问数据（必填，需要 internal_base_salary 字段）
  - ❌ 费用数据（不需要）

📈 **计算方式**：
  - 顾问成本 = internal_base_salary × 3 × 1.2
    * 3倍包括：工资、社保公积金、顾问奖金、增值税、营运成本
    * 1.2 = 20%利润目标
  - ❌ 不包含真实运营费用明细

🎯 **用途**：给顾问展示的盈亏分析，避免真实数据泄露

💡 **提示**：内部核算版本只需要成单数据和顾问数据即可计算，无需上传费用表！
