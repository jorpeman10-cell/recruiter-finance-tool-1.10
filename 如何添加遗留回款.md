# 如何添加上年遗留回款

## 问题
如果你的数据已经上传但显示收入为0，可能是**缺少上年遗留回款列**。

## 解决方案

### 方法1：在原有CSV中添加列
在你的成单数据CSV文件中，添加一列名为 `prior_year_collection` 或 `上年遗留回款`：

**修改前：**
```csv
deal_id,client_name,candidate_name,position,consultant,deal_date,fee_amount,payment_status,actual_payment
D001,客户A,张三,职位1,顾问A,2026-01-15,500000,未回款,0
D002,客户B,李四,职位2,顾问B,2026-02-20,500000,已回款,116000
```

**修改后：**
```csv
deal_id,client_name,candidate_name,position,consultant,deal_date,fee_amount,payment_status,actual_payment,prior_year_collection
D001,客户A,张三,职位1,顾问A,2026-01-15,500000,未回款,0,1000000
D002,客户B,李四,职位2,顾问B,2026-02-20,500000,已回款,116000,950000
```

### 方法2：使用中文列名
也可以使用中文列名，系统会自动识别：

```csv
成单编号,客户名称,候选人,职位,负责顾问,成单日期,佣金,回款状态,实际回款,上年遗留回款
D001,客户A,张三,职位1,顾问A,2026-01-15,500000,未回款,0,1000000
D002,客户B,李四,职位2,顾问B,2026-02-20,500000,已回款,116000,950000
```

## 数据逻辑说明

假设你的数据情况：
- 本年Offer（总佣金）：150万
- 本年已回款：11.6万  
- 去年遗留回款：195万
- 本年累计成本：195万

**计算公式：**
- 本年收入合计 = 本年已回款 + 上年遗留回款 = 11.6万 + 195万 = 206.6万
- 利润 = 本年收入合计 - 成本 = 206.6万 - 195万 = 11.6万

## 检查数据是否加载成功

上传数据后，在侧边栏"数据概览"部分可以看到：
- ✅ 已加载 X 条成单数据
- 总佣金: ¥XXX
- 本年回款: ¥XXX
- 上年遗留回款: ¥XXX （如果有）
- 本年收入合计: ¥XXX

如果"上年遗留回款"显示为0，说明该列数据没有被正确读取，请检查列名是否正确。
